About

HTML Online Viewer is a fast HTML editor and formatter with an instant live preview. View, edit and format your HTML in real-time!

You can see the live preview of the HTML on the same page, but you can also show the preview in a new separate tab where it is more stable.

The viewer highlights the HTML for better readability using the high performance editor: Ace (license). The highlighted code can also be shown in a separate tab where it is rendered at once (not in the editor), so the browser's native search function can be used (Ctrl+F). This highlighting is done using a reliable third-party library: highlight.js (license).

The formatting function can be useful too. It is done using an efficient third-party library: JS Beautifier (license).

Available functions:

  • Preview (Alt+1) button renders the HTML code in a new tab.
  • Highlight (Alt+2) button shows the entire highlighted code in a new tab.
  • Format (Alt+3) button beautifies the HTML text.
  • Expand all (Alt+4) button expands all elements.
  • Collapse all (Alt+5) button collapses all elements.
  • Sample (Alt+6) button puts a sample HTML text into the editor.
  • Clear (Alt+7) button erases the editor text.

This website uses cookies for traffic monitoring and running ads. If you don't want to allow cookies, you can disable them in Privacy settings.

All input is processed locally (not by a server) and it is thus 100% safe and private.

function populateDashboard() {
document.getElementById('stats-companies').textContent = companiesData.length;
document.getElementById('stats-countries').textContent = [...new Set
                    (companiesData.map(c => c.country))].length;
document.getElementById('stats-contacts').textContent = companiesData.reduce((acc
                    , c) => acc + (c.contacts ? c.contacts.length : 0), 0);
const contactsList = document.getElementById('contacts-list');
contactsList.innerHTML = ''; // Clear existing contacts
const contactsWithCompany = companiesData
.filter(c => c.contacts && c.contacts.length > 0)
.flatMap(c => c.contacts.map(contact => ({...contact, companyName: c.name}
                        )));
contactsWithCompany.forEach(contact => {
const card = document.createElement('div');
card.className = 'bg-white p-4 rounded-lg shadow hover:shadow-xl transition
                        -shadow';
card.innerHTML = `
<h4 class="font-bold text-lg text-[#36454F]">${contact.name}</h4>
<p class="text-sm text-[#4682B4]">${contact.title}</p>
<p class="text-sm text-gray-500 mt-1">${contact.companyName}</p>
`;
contactsList.appendChild(card);
});
const nonAlcoholicDiv = document.getElementById('non-alcoholic-products');
nonAlcoholicDiv.innerHTML = ''; // Clear existing
const alcoholicDiv = document.getElementById('alcoholic-products');
alcoholicDiv.innerHTML = ''; // Clear existing
companiesData.forEach(c => {
const suitabilityNote = c[`suitability_notes_${currentLang}`] || '';
const adaptationNote = c[`adaptation_notes_${currentLang}`] || '';
if(c.products_non_alcoholic && c.products_non_alcoholic.length > 0) {
const item = document.createElement('div');
item.className = 'text-sm p-2 bg-green-50 rounded';
item.innerHTML = `<strong>${c.name}:</strong> ${c.products_non_alcoholic
                            .slice(0,3).join(', ')}... <em class="text-xs block text-green-800"
                            >${translations[currentLang].modalNote} ${suitabilityNote}</em>`;
nonAlcoholicDiv.appendChild(item);
}
if(c.products_alcoholic && c.products_alcoholic.length > 0) {
const item = document.createElement('div');
item.className = 'text-sm p-2 bg-red-50 rounded';
item.innerHTML = `<strong>${c.name}:</strong> ${c.products_alcoholic.join
                            (', ')} <em class="text-xs block text-red-800"
                            >${translations[currentLang].modalAdaptationNote} ${adaptationNote}
                            </em>`;
alcoholicDiv.appendChild(item);
}
});
}
